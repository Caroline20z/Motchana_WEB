Назва атрибуту: hidden

Являє собою вимикач, який виявляє, відображати об'єкт у вікні браузера чи ні. Це особливо зручно для приховання панелі керування при відтворені фонової музики. Якщо цей атрибут вказан, значення атрибутов width і height ігнорується.


Синтаксиc:

< embed hidden="true | false" >


Значення:


Значення Короткий опис
true Приховує об'єкт
false Відображає об'єкт на сторінці
Значення за замовчуванням

InternetExplover Chrome Opera Safari Firefox Android Apple
3.0+ 1.0+ 4.0+ 1.0+ 1.0+ 1.0+ 1.0+
